# DemoWebflux ``Portfolio project``
Java backend restfull reactive application

# Requirements:
- Java 17 or above
- Apache Maven 3.6.3 or above

# Technologies

## Backend
* Spring Boot
* Spring WebFlux
* R2DBC

### Databases
#### SQL
* PostgreSQL

#### Migrations
* Flyway

### Code Write
* Lombok
* Mapstruct

### Services
* Github actions
* Docker & Docker-compose
